# Point Distance Calculator

## pre-requirements

- [poetry](https://python-poetry.org/)
- Python 3.10.12 (see [.pythin-version](./.python-version)

## setup

```shell
poetry install
```

## test

```shell
poetry run pytest
```

## execute

```shell
poetry run main
```
